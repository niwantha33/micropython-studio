# Code of Conduct

## Our Promise

We want MicroPython Studio to be a friendly, welcoming place for everyone — regardless of experience level, background, or identity.

## The Rules

**Be kind.** That's the main rule. Everything else follows from it.

### Do

- Be respectful and constructive in discussions
- Welcome newcomers and help them get started
- Give credit where it's due
- Accept feedback gracefully
- Focus on what's best for the project and community

### Don't

- Insult, troll, or put others down
- Harass anyone for any reason
- Post spam or off-topic content
- Share others' private information
- Do anything you wouldn't want done to you

## Reporting Issues

If someone is breaking these rules, email **niwantha33@gmail.com**. All reports will be handled privately and taken seriously.

## What Happens

1. **First time** — A friendly reminder
2. **Repeated** — Temporary ban from the project
3. **Serious or ongoing** — Permanent ban

## Scope

This applies everywhere the project lives: GitHub issues, pull requests, discussions, and any related community spaces.
